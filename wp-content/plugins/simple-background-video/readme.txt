== Description ==

[Simple Background Video](http://www.zuhaitz-design.com/blog/simple-background-video) You will be able to add a title and a video in full screen mode in any page of your website and even in your Homepage with "Simple Background Video". Allowed video formats are Youtube, Vimeo and Mp4. You have the option to put a title, and in that case you could put any title in any color.

**All Features | Everything is free**

* Add a Background Video at Home
* Add a background video to any page
* Put a Custom title
* Change the color of the title
* Youtube video
* Vimeo video
* Upload MP4 format
* Image for mobil/tablet version


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/simple_background_video_zd` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Simple Background Video Settings screen to configure the Background Video Home
4. Add a background video to any page in "individual Edit Page".
