== Description ==

Visual pizza builder for WooCommerce.

Features:

* WooCommerce integration
* 25 premade ingredients
* Customisable ingredients
* Left and right parts are customisable
* Regular/extra portions
* Photo and description of ingredients
* Maximum number of ingredients limit
* Pizza image adjusts on scroll
* Responsive layout
* Translation ready

Demo http://wppizzatime.com/product/custom-pizza-demo/

Pro version of the plugin has all of the above features plus:

* Extra color schemes 
* Pizza presets 
* Custom sections 

== Installation ==

* Copy pizzatime folder to wp-content/plugins or upload the zip file through "Plugins -> Add New" page.
* Activate the plugin from the Plugins menu within the WordPress admin.
* On the settings page configure the main settings and ingredients.
* Check how to create a PizzaTime product on the settings page.
